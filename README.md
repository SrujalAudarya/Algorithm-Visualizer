# Algorithm Visualizer

An interactive web application that visualizes sorting and pathfinding algorithms step by step in real time. Built with React.js and deployed on Vercel.

🔗 **Live Demo:** https://algorithm-visualizer-ten-pied.vercel.app/

---

## What it does

- **Sorting Visualizer** — Watch algorithms sort an array with animated bar comparisons and swaps
- **Pathfinding Visualizer** — Draw walls on a grid and watch algorithms find the shortest path

---

## Algorithms Included

**Sorting**
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Heap Sort

**Pathfinding**
- Breadth-First Search (BFS)
- Depth-First Search (DFS)
- Dijkstra's Algorithm
- A* Search

---

## Tech Stack

- React.js
- JavaScript (ES6+)
- HTML5 & CSS3
- Deployed on Vercel

---

## How to Run Locally

```bash
# Clone the repository
git clone https://github.com/SrujalAudarya/Algorithm-Visualizer.git

# Go into the project folder
cd Algorithm-Visualizer

# Install dependencies
npm install

# Start the app
npm start
```

Open http://localhost:3000 in your browser.
